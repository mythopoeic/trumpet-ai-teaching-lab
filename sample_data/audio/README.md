# Sample Audio

This folder intentionally contains **no audio recordings**.

The audio analysis components (spit-buzz analyzer, double-pedal detector,
played-passage tone analyzer) are exercised in the test suite with **synthetic,
programmatically generated signals**, not real recordings. Real practice
recordings, lesson recordings, and any reference clips used during development
are private and are excluded from this public repository (see the
"Rights and Privacy" section of the top-level README).

If you add sample audio here, use only **synthetic or rights-cleared** clips and
document their provenance.
