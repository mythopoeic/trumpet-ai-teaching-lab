# Era system prompt (trumpet-yoga) — skeleton

Production prompt content for the **trumpet-yoga** era is withheld in this public
portfolio snapshot. The real prompts encode proprietary, Jerome Callet-inspired
teaching behavior and are not published.

Structure (what the production prompt provides, in order):

1. **Role** — the assistant's teaching persona and scope.
2. **Era context** — the concepts and vocabulary specific to this era.
3. **Source-grounding rules** — answer only from retrieved, cited source
   material; flag uncertainty; never fabricate.
4. **Response format** — how explanations, citations, and any audio feedback are
   shaped for the student.
